Python 語言擁有清晰易懂的句法及架構，很適合作為教學工具。Python 主要用於非圖像式的程式應用，因為它本身沒有屬於自己的圖像程式庫，但是可以挪用某些跨平台的圖像工具，製作圖像應用程式。

Python 語言是整個社群的心血結晶，並不是商業計畫。由好心的創始者 Guido van Rossum 發起。Python 的特色在於句法少，但是擁有許多五花八門模組(module)，以延伸語言功能。您可以在此網址 http://www.python.org 獲得更多相關資訊。
