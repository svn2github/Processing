Processing was created to both teach fundamentals of computer programming within a visual context and to be used as a production tool. This section of the website presents a few series of short examples to get people of all skill levels started with using Processing. The books listed on the right have been written primarily for beginners and intermediate-level programmers

Basics
Introducing the basic elements of computer programming and the fundamental components of Processing.

Topics
Areas such as animation, drawing, interaction, motion, simulation, file i/o, cellular automata, fractals, and l-systems.

3D & OpenGL
Processing has two 3D renderers that can render 3D shapes on screen and control lighting and camera parameters. The P3D renderer is an optimized software renderer and the OPENGL renderer uses JOGL to access OpenGL accelerated graphics cards (this creates an enormous speed improvement on computers with supported graphics cards.)

Libraries
The libraries built for Processing extend the software beyond graphics. They enable the software to capture and playback video, import SVG files, export PDF files, communicate using the Internet and RS-232 protocols, create and playback sound files, and more...
