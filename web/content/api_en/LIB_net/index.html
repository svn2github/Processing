The Net library makes it easy to read and write data across machines on the Internet. It allows the creation clients and servers. A server connects to a list of clients for reading and writing data. A client is able to read and write data to a server.

Server
The server class is used to create server objects which send and receives data to and from its associated clients (other programs connected to it).

Server
write()
available()
stop()
disconnect()

serverEvent()


Client
The client class is used to create client Objects which connect to a server to exchange data.
